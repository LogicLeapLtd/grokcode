The grok CLI cold-boots its MCP servers on every run — long enough to glance at Slack and lose your train of thought. GrokCode keeps **one grok session warm all day**, so the first token lands in **under 200ms**. Same CLI, same keys, nothing leaves your Mac.

## What you get

- 🔥 **Always-warm session** — no cold-start wait, ever. One `grok agent stdio` session, booted once, kept alive.
- ✍️ **Real markdown chat** — headings, tables, one-click-copy code blocks, and a collapsible reasoning trace.
- ⌨️ **⌘K everything** — jump to any project, chat, or action; slash-commands and `@`-file mentions drop context straight into the prompt.
- 🧩 **Plugin marketplace** — one-click MCP installs (GitHub, Supabase, Playwright, Notion, Linear…). Build your own and publish it from inside the app.
- 🤖 **Automations** — save any prompt and run it manually, on an interval, or daily.
- 🌙 **Native macOS** — dark mode throughout, keyboard-first, fast.
