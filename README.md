## 🟢 It runs on your SuperGrok subscription

No xAI API key. No per-token meter. **GrokCode uses the SuperGrok sub you already pay for** — so you can prompt as much as you want and the bill never moves. Every other Grok GUI makes you plug in an API key and pay *again* per token. This one doesn't.
